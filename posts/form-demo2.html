发表于2022-01-06|更新于2023-04-26|HTML
|阅读量:

相遇皆是缘分

概述

实现简单的邮箱验证及样式效果

页面效果

html代码

<div class="container">
	<input type="email" name="" id="mail" required placeholder="输入邮箱">
	<span class="title">邮箱</span>
	<label for="mail"></label>
</div>

css代码

<style>
	.container{
		margin: 100px;
		/* 相 对 定 位 */
		position: relative;
	}
	input{
		border: 1px solid #999;
		outline: 0;
		width: 140px;
		height: 30px;
		line-height: 30px;
		/* 文 字 缩 进 */
		text-indent: 36px;
		/* 过 渡 效 果 */
		transition: all .3s;
		border-radius: 5px;
	}
	span.title{
		/* 绝 对 定 位 */
		position: absolute;
		line-height: 30px;
		height: 30px;
		left: 2px;
		top: 2px;
		transition: all .3s;
	}
	input:focus,input:hover{
		text-indent: 2px;
	}
	input:focus + span.title , input:hover + span.title{
		transform: translateX(-120%);
	}
	input:valid ~ label::after{
		content: "邮箱输入正确";
	}
	input:invalid ~ label::after{
		content: "邮箱输入错误";
	}
	input:valid{
		border: 1px solid green;
	}
	input:invalid{
		border: 1px solid red;
	}
</style>
